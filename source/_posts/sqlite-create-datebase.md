其实很简单，但刚开始一点都不会。
```shell
$sqlite test.db
SQLite version 2.8.17
Enter ".help" for instructions
sqlite> .database
seq  name             file                                                      
---  ---------  -------------------------
0    main        /home/pi/code/testDB.db
```
这样就创建成功了。
